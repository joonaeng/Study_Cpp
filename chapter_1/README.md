# Chapter 1 - C언어 기반의 C++

## 학습 내용 요약

- `std::cin`, `std::cout`, `std::endl` → 입력, 출력, 줄바꿈
- `namespace` → 같은 이름 구분
- `using namespace std;` → std 생략
- 함수 오버로딩 → 같은 이름, 다른 매개변수로 여러 함수 작성
- 매개변수 디폴트값 → 오른쪽부터 기본값 설정
- 인라인 함수 → 실행속도 향상, 작은 함수에 적합
